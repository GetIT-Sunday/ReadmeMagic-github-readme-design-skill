🚀 Quick Start

# Install
pip install github-readme-design-skill

# Generate README
github-readme-design generate --project-path ./my-project

# Use specific template
github-readme-design generate --template ai-project --project-path ./my-ai-project

# Preview result
github-readme-design preview --output preview.html
